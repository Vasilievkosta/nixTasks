Практика Advanced JS 4.

Дан массив с числами, в нем необходимо найти и поместить в массив индексы двух чисел, сумма которых будет равна заданному числу N.

Если таких чисел нет, то вернуть пустой массив.

Пример кода:
example([2, 7, 11, 15], 9) => [0, 1]
example([12, 24, 34, 2], 14) => [0, 3]
example([2, 7, 11, 15], 22) => [1, 3]

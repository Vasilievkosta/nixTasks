Практика Advanced JS 8.

У нас есть время, но с ним что-то пошло не так! Я смотрю на часы, а они показывают 76 минут пятого!

Исправьте это недоразумение, пожалуйста. Примеры ниже.

Пример кода:
timeCorrect('09:10:01') => '09:10:01'
timeCorrect('11:70:10') => '12:10:10'
timeCorrect('19:99:09') => '20:39:09'
timeCorrect('19:99:99') => '20:40:39'
timeCorrect('24:01:01') => '00:01:01'
timeCorrect('52:01:01') => '04:01:01'

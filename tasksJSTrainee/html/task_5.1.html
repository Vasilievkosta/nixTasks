Практика Advanced JS 5.1.

Необходимо написать функцию, которая принимает строку в качестве аргумента и выводит количество гласных в этой строке.

Использовать алфавит русского языка. Необходимо написать как минимум 3 реализации функции.

Вариант 1.

Решение использует конструкцию str.match, но она может вернуть null, поэтому проверить через тернарный оператор.

Пример кода:
findVowels('Привет') => 2
findVowels('Абракадабра') => 5

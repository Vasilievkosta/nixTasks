Практика JS 18.

Никаких разговоров. Никаких объяснений. Только ТЗ и результат, который должен быть. Посмотрите, что должно получиться на выходе и напишите эту функцию.

Пример кода:
spacey(['kevin', 'has','no','space'])
возвращает => [ 'kevin', 'kevinhas', 'kevinhasno', 'kevinhasnospace']
spacey(['this','cheese','has','no','holes'])
возвращает => ['this','thischeese','thischeesehas','thischeesehasno','thischeesehasnoholes']

Практика JS 16.

Напишите функцию example, которая принимает в себя строку и число. Верните, в виде массива, только те слова, длинна которых превышает число.

Пример кода:
example('Сегодня отличный день. С самого утра мы пойдем на озеро, купаться.', 5)
а возвращает => ['Сегодня', 'Отличный', 'самого', 'пойдем', 'купаться']
example('Над городом облака, коридоры, берега, проспект, река.', 7)
а возвращает => ['коридоры', 'проспект']
